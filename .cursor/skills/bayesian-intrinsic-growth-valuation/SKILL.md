# 贝叶斯内在增长估值

## 核心原则

不要将公司新闻简单归类为利好或利空。将每条公司相关数据点转化为对未来 3-5 年收入增速、利润率、TAM、市占率、估值倍数和市场情绪的概率更新。

目标是估算公司真实内在增长速度，并与当前市值已隐含的增长进行比较。

将输出视为研究假设，而非个性化投资建议。在做出时效性判断前，务必从可靠当前来源核实市值、价格、收入、利润率、filings、指引、同行倍数和新闻。

## 所需输入

使用用户提供的任何数据，并明确标注需要核实的缺失变量：

- 公司基本面：收入规模、利润率、自由现金流、ROIC、资产负债表、客户、护城河、定价权
- 行业周期：需求增速、供需缺口、库存周期、订单周期、价格趋势、政策、下游 capex
- 营收与增长：历史增速、指引、backlog、book-to-bill、有机增长、ASP、出货量
- TAM 与 TAM 增速：当前 TAM、未来 TAM CAGR、渗透率、市占率、新市场扩张
- 估值：EV/Sales、EV/EBITDA、P/E、FCF yield、PEG、历史分位、同行分位、隐含增长
- 股价走势：1M/3M/6M/12M 及财报后回报、回撤/反弹路径、波动率、成交量、相对行业/指数表现，以及股价上涨是否领先于内在增长
- 市场 FOMO：股价变动、期权活动、社交热度、分析师上修、主题拥挤度、叙事强度
- 新信息：订单、客户、产品、定价、政策、竞争、产能、财报、管理层指引

## 增长假设

始终将未来 3-5 年收入 CAGR 表述为以下假设的概率分布：

| 假设 | 标签 | 3-5 年收入 CAGR |
| --- | --- | --- |
| H0 | 衰退型 | <0% |
| H1 | 低速成熟 | 0%-5% |
| H2 | 稳定成长 | 5%-12% |
| H3 | 高景气成长 | 12%-25% |
| H4 | 结构性爆发 | 25%-50% |
| H5 | 平台级扩张 | >50% |

## 工作流程

### 1. 建立先验

根据基本面、行业周期、TAM、历史增速和竞争地位，为 H0-H5 分配初始概率。

证据不完整时优先保守先验。不要让市场兴奋单独支撑 H4 或 H5。

### 2. 按变量分类新信息

出现新信息时，识别其影响的变量：

- 收入增速
- 利润率
- TAM
- 市占率
- 竞争结构
- 现金流
- 估值倍数
- FOMO 情绪

若信息主要影响市场关注度，更新估值倍数和 FOMO，而非内在增长。

### 3. 贝叶斯更新

追问该信息在各增长假设下出现的可能性：

- 若更符合 H3/H4/H5，提高这些概率。
- 若似周期性、一次性或 backlog 时序因素，避免过度更新长期增长。
- 若仅强化叙事或交易热情，提高 FOMO 和倍数风险，而非内在增长。
- 若与高增长矛盾，将概率向 H0-H2 转移。

展示更新过程：先验 -> 似然解读 -> 后验。

### 4. 计算加权内在增长

根据后验概率估算加权内在 3-5 年收入 CAGR。除非有更好证据，使用区间中点假设：

| 假设 | 建议中点 |
| --- | --- |
| H0 | -5% |
| H1 | 2.5% |
| H2 | 8.5% |
| H3 | 18.5% |
| H4 | 37.5% |
| H5 | 60% 或情景特定值 |

报告区间，避免虚假精度。

### 5. 反推市场隐含增长

根据当前市值或企业价值、收入、利润率、FCF margin、估值倍数和折现率假设，推断当前估值嵌入的增长率。

若缺少精确数据，说明缺失输入，给出定性隐含增长区间，而非编造数字。

### 6. 比较内在增长与市场隐含增长

分类估值状态：

| 比较 | 估值状态 |
| --- | --- |
| 内在增长 > 隐含增长 | 低估 |
| 内在增长 ≈ 隐含增长 | 合理 |
| 隐含增长 > 内在增长，但周期仍加速 | 高估但可交易 |
| 隐含增长远高于内在增长且 FOMO 极端 | 泡沫化 |

### 7. 衡量股价-增长背离

单独判断股价趋势相对内在增长更新是更快还是更慢。

尽可能使用当前数据：

- 比较近期股价回报、市值扩张和倍数扩张与收入 CAGR、指引、backlog、利润率和后验概率的变化
- 区分由基本面驱动的重估与由流动性、主题拥挤、轧空、指数资金流或 FOMO 驱动的重估
- 将背离分类为 `股价落后基本面`、`股价匹配基本面`、`股价领先基本面` 或 `严重股价-增长背离`
- 股价领先内在增长时，即使公司质量仍高，也应降低长期安全边际信心
- 股价落后内在增长时，识别市场弥合缺口所需的催化剂

建议定性阈值：

| 股价变动 vs 内在增长更新 | 背离信号 |
| --- | --- |
| 股价回报明显低于改善的后验增长 / 隐含增长仍低于内在增长 | 股价落后基本面 |
| 股价回报和倍数扩张大致匹配后验增长改善 | 匹配 |
| 股价回报或倍数扩张超过后验增长改善 | 股价领先基本面 |
| 股价快速上涨、倍数重估，但 posterior 内在增长几乎无改善 | 严重背离 / FOMO 风险 |

### 8. 构建验证路径

定义验证或证伪模型的时间窗口和具体指标：

- 收入增长和指引修订
- backlog、book-to-bill、订单、交期
- ASP、出货量、产能利用率、扩产
- 毛利率、经营杠杆、FCF 转化
- TAM 扩张证据和渗透率变化
- 市占率得失
- 同行/客户/供应商佐证
- 分析师修正广度和叙事拥挤度

## 输出模板

公司分析使用以下格式：

```markdown
## 1. 公司一句话定位
说明公司到底是什么，以及增长由什么驱动。

## 2. 当前增长假设概率表
| 假设 | CAGR 区间 | 先验概率 | 更新后概率 | 核心理由 |
| --- | --- | ---: | ---: | --- |
| H0 衰退型 | <0% |  |  |  |
| H1 低速成熟 | 0%-5% |  |  |  |
| H2 稳定成长 | 5%-12% |  |  |  |
| H3 高景气成长 | 12%-25% |  |  |  |
| H4 结构性爆发 | 25%-50% |  |  |  |
| H5 平台级扩张 | >50% |  |  |  |

## 3. 加权内在增长速度
给出未来 3-5 年收入 CAGR 的加权区间和关键假设。

## 4. 市场隐含增长速度
反推当前市值/估值倍数隐含的增长率；若数据不足，列出需要补齐的数据。

## 5. 股价走势与内在增速背离
比较 1M/3M/6M/12M 股价、相对行业/指数表现、市值和估值倍数变化，与收入增速、指引、订单、利润率和 posterior 增长概率变化是否匹配。
给出结论：股价落后基本面 / 股价基本匹配基本面 / 股价领先基本面 / 严重背离且 FOMO 风险上升。

## 6. 新信息的贝叶斯更新
说明信息影响的变量、在各增长假设下的相容性，以及 posterior 变化。

## 7. 估值状态
在 低估 / 合理 / 高估但可交易 / 泡沫化 中选择一个，并解释为什么。

## 8. 上行空间
说明需要哪些收入、利润率、TAM、市占率或倍数条件才有上行。

## 9. 下行风险
列出增长、利润率、竞争、周期、估值、FOMO 和流动性风险。

## 10. 验证周期
说明应在几个季度内验证，以及每个阶段看什么。

## 11. 关键跟踪指标
列出最重要的财报、订单、价格、产能、客户、股价相对表现、成交量、波动率、估值分位和情绪指标。

## 12. 仓位建议
用观察 / 小仓试错 / 验证后加仓 / 只交易不投资 / 降级或退出 等条件化表述，避免个性化投资指令。

## 13. 一句话结论
用一句话总结内在增长、市场隐含增长与股价走势之间的差异。
```

## 风格规则

- 从可观察的需求变化出发，而非表面叙事。
- 将需求翻译为收入、利润、TAM 和估值影响。
- 寻找被低估的卖铲子、瓶颈、二号位、硬制造和关键供应链节点。
- 区分内在增长更新与 FOMO 和倍数扩张。
- 明确衡量股价变动是领先、匹配还是落后内在增长更新。
- 区分结构性增长与周期反弹或一次性订单时序。
- 明确说明不确定性、缺失数据和证伪条件。
