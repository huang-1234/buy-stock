# GF-DMA 健康指数

## 核心思想

评估股价当前趋势是否由基本面速度和均线结构共同支撑。

使用本指数回答：

```text
当前股价趋势是否被营收增速、利润增速、预期上修和 20/50/100/200DMA 系统共同承接？
```

将结果视为研究分析，而非投资建议。评分前务必从当前来源核实最新数据。

## 所需输入

评分前收集最新可用数据：

- 价格/技术数据：最新股价、20DMA、50DMA、100DMA、200DMA、ATR20、5 日股价变化，以及 20/50/100/200 日价格变化或历史价格。
- 基本面数据：最新季度营收、EPS、毛利率或毛利润、下一季度公司指引、一致预期营收/EPS，以及 30 天预期修正。
- 优先来源：公司 IR 发布/演示、财报电话、Yahoo Finance 历史价格/分析、TradingView 技术面/预期、Barchart 技术分析、Seeking Alpha 预期、Koyfin、FactSet、Bloomberg、TIKR 或 Visible Alpha。

若缺少必填字段，说明缺失项；仅在适当时使用简化公式。

## 计算流程

### 1. 基本面速度

计算：

```text
G_f = 0.35G_Revenue + 0.25G_GrossProfit + 0.30G_EPS + 0.10G_Revision
```

其中：

- `G_Revenue = 下一季度营收指引 / 最新季度营收 - 1`
- `G_GrossProfit = 下一季度毛利润 / 最新季度毛利润 - 1`
- `G_EPS = 下一季度 EPS 指引 / 最新季度 EPS - 1`
- `G_Revision = 30 天一致预期修正幅度`

降级方案：

- 若缺少毛利润或 EPS：`G_f = 0.5G_Revenue + 0.5G_EPS`
- 若仅有营收指引：`G_f = G_Revenue`

### 2. DMA 速度

计算每条均线的季度化等效斜率：

```text
G_DMAx = ((SMA_x(t) - SMA_x(t-k)) / SMA_x(t-k)) * (63 / k)
```

默认 `k = 5` 或 10 个交易日。若仅有价格变化数据，可近似：

```text
DailySlope_x ~= (P_t - P_t-x) / x
G_DMAx ~= DailySlope_x * 63 / P_t
```

计算 `G_DMA20`、`G_DMA50`、`G_DMA100` 和 `G_DMA200`。

### 3. 基本面-DMA 匹配度

计算：

```text
R_x = G_DMAx / G_f
```

优先解读 `R_50` 和 `R_100`：

| R_x | 状态 |
| ---: | --- |
| < 0.5 | 趋势明显低于基本面速度 |
| 0.5-0.8 | 相对趋势未充分反映或偏便宜 |
| 0.8-1.3 | 健康匹配 |
| 1.3-2.0 | 偏热但可能可解释 |
| > 2.0 | 过热 / FOMO 逃逸风险 |

核心均线侧重：

| 股票类型 | 关键均线 |
| --- | --- |
| NVDA、AVGO、MSFT 等大盘成长股 | 50DMA |
| MU、SNDK 等存储/周期半导体 | 100DMA |
| LITE、AAOI 等高弹性光模块 | 20DMA + 50DMA |
| ETN、VRT、TEL 等工业 AI/电力 | 100DMA + 200DMA |
| SIVE、CPSH 等小市值硬制造 | 20DMA + ATR 背离 |
| SOXX、SMH 等半导体 ETF | 50DMA + 100DMA |

### 4. 股价-均线背离

计算：

```text
D_x = P_t / SMA_x(t) - 1
Z_x = (P_t - SMA_x) / ATR20
```

解读：

| 信号 | 状态 |
| --- | --- |
| 高于 20DMA 0%-5% | 健康贴线趋势 |
| 高于 20DMA 5%-12% | 强趋势 |
| 高于 20DMA 12%-20% | 偏热 |
| 高于 20DMA >20% | 短线逃逸 |
| 高于 50DMA >30% | 中期过热 |
| 高于 100DMA >50% | 大级别重估 |
| 高于 200DMA >100% | 长周期极端重估 |

ATR 背离：`Z_x` 为 0-2 健康，2-3 偏热，3-4 高热，>4 逃逸，<0 表示股价低于该均线。

### 5. 趋势平行度 / 逃逸比率

计算：

```text
EscapeRatio = 5 日股价斜率 / 50DMA 日斜率
```

解读：

| EscapeRatio | 状态 |
| ---: | --- |
| 0.8-1.2 | 股价与 50DMA 平行，健康 |
| 1.2-1.8 | 短期加速，可接受 |
| 1.8-2.5 | 明显偏热 |
| >2.5 | FOMO 逃逸 |
| 0-0.5 | 动能衰减 |
| <0 | 短线反转，趋势受损 |

### 6. 预期上修确认

预期修正评分：

| 修正状态 | 分数 |
| --- | ---: |
| 营收和 EPS 预期持续上修，公司指引高于市场 | 85-100 |
| 温和上修，指引略高于市场 | 70-85 |
| 预期稳定，缺少进一步上修 | 55-70 |
| 预期开始下修 | 35-55 |
| 指引低于市场，分析师集中下修 | <35 |

## 最终评分

计算 100 分制总分：

```text
HealthScore = 40S_GrowthMatch + 25S_Divergence + 20S_Parallel + 15S_Revision
```

模块权重：

| 模块 | 权重 |
| --- | ---: |
| 基本面速度匹配 | 40% |
| 股价-均线背离 | 25% |
| 趋势平行度 | 20% |
| 预期上修确认 | 15% |

最终解读：

| 分数 | 状态 | 含义 |
| ---: | --- | --- |
| 85-100 | Healthy Momentum | 健康主升浪 |
| 75-85 | Strong but Watch | 强趋势，继续观察 |
| 65-75 | Hot but Supported | 偏热，基本面仍可支撑 |
| 55-65 | Damaged / Overheated | 趋势受损或局部过热 |
| 40-55 | High Risk | 风险明显上升 |
| <40 | Broken / Escaping | 趋势破坏或逃逸后回落 |
