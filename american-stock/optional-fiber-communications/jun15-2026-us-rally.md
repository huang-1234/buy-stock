## 2. 今晚为什么涨？——四条驱动

### 2.1 宏观：美伊和谈 → 全球 risk-on（~40%）

6 月 14–15 日，美伊宣布 **停战框架 / MOU**，霍尔木兹海峡有望重开：

- 原油单日约 **-5%**，通胀担忧降温  
- 亚太股市大涨（日经 +5%+、韩股 +6%+）  
- 美股 6/15 标普 **+0.5%～+1.5%**，成长股、高 Beta 科技受益  

LITE / COHR / AAOI **Beta 高**（AAOI Beta ~3.7），与半导体、AI 基建一起被 **「风险偏好回升」** 带动——**涨的是情绪，不是新订单**。

### 2.2 板块：6/9 暴跌后的修复性反弹（~30%）

6 月 9 日 SemiAnalysis 报告「CPO 推迟到 2028–2029」引发恐慌性抛售：

| 6/9 单日跌幅 |
| --- |
| AAOI **-17%** |
| COHR **-15%** |
| LITE **-12%** |
| MRVL **-13%** |

6/15 是在 **「先砸再弹」** 框架里的修复：

- 英伟达 Gilad Shainer：**Spectrum-X CPO 2026 下半年按计划交付**  
- 黄仁勋 Computex：**「必须用光的地方用光」**（copper where you can, optics where you must）  
- 市场从「CPO 没了」切回「CPO 晚两年 + 可插拔 800G/1.6T 照涨」

### 2.3 AAOI 个股：Spectrum QuantumLink 软件合作（~15%）

AAOI **+15%** 领跑，不全是板块 Beta：

- 与 **Spectrum** 扩大 **QuantumLink** 智能远程管理平台部署  
- 从纯硬件光模块 → **软件 + recurring 收入** 叙事  
- 小盘高弹性，6/9 跌深后 **空头回补 + 动量资金** 反扑  

### 2.4 结构：Basket 交易 + AI 光互联老逻辑（~15%）

LITE / COHR / AAOI / GLW / MRVL 常被当作 **同一篮子**：

- Hyperscale capex 超预期  
- NVDA 对 LITE、COHR 各 **$2B** 投资 + 采购承诺  
- Laser 供给缺口 >30%（LITE 管理层口径）  
- 800G → 1.6T 升级周期  

**一条利好 → 全板块一起动**，6/15 就是这种联动。

### 2.5 驱动权重

```text
美伊和谈 risk-on          ████████░░  ~40%
6/9 暴跌后叙事修复         ███████░░░  ~30%
AAOI Spectrum 软件故事     ████░░░░░░  ~15%（主要推 AAOI）
```

**结论：6/15 主要是「宏观情绪 + 板块修复」，不是三家同时发布新财报或大额订单。**

## 5. 6/16（周二）走势判断

> 基于 GF-DMA 框架 + 该板块历史波动（6/9 先例：大涨后次日可 -8%～-15%）。

### 5.1 基准情景（~55%）：震荡分化，小跌或平盘

| 依据 | 说明 |
| --- | --- |
| 和谈是 **已知利好** | 6/15 已涨，易 **「利好出尽」** |
| 6/9 先例 | 大涨后次日可 **-8%～-15%** |
| 估值已贵 | LITE YTD +166%，COHR +97%，AAOI +440%+ |
| LITE Escape 偏高 | 短线 FOMO 后常 **1–3 日整理** |
| 无新财报 | 下一验证 ~8 月 Q4 |

**典型走法：** 高开 → 获利盘 → 收盘 **±2%～±4%**，AAOI 波动 > COHR > LITE。

### 5.2 偏强情景（~25%）：续涨 +3%～+8%

需同时满足：

- 标普/纳指延续 risk-on  
- MRVL、NVDA、GLW 跟涨  
- 和谈无反复、无新 CPO 看空报告  
- 成交量 **明显大于** 5 日均量  

→ 可能再冲，但 **难复制 AAOI +15%**。

### 5.3 偏弱情景（~20%）：回调 -5%～-12%

触发条件：

- 和谈细节生变（以色列/黎巴嫩等）  
- 原油反弹、通胀预期回升  
- 新的 CPO / AI capex 看空研报  
- 大盘 risk-off  

→ 光模块 **Beta 高，往往跌得比大盘多**；AAOI 首当其冲。

### 5.4 概率表

| 情景 | 概率 | LITE | COHR | AAOI |
| --- | ---: | --- | --- | --- |
| 震荡 ±2%～4% | **~55%** | 最稳 | 中等波动 | 波动最大 |
| 续涨 +3%～+8% | ~25% | 需大盘配合 | 可能领涨 | 弹性最大 |
| 回调 -5%～-12% | ~20% | 贴 20DMA ~$905 | 看 $385 支撑 | 看 $176 低点 |

**一句话：6/16「再大涨」和「大跌」都不是最高概率；更可能是涨完后的整理日，AAOI 振幅最大。**

### 9.2 为什么 6/16 大跌？——四条原因

#### ① 利好出尽 / 卖事实（~35%）

6/15 涨的主因是 **美伊和谈 risk-on**（见 §2.1）。6/16 CNBC 报道：

→ **Day 2 卖事实**：和谈 premium 在 6/15 已 price-in，6/16 资金兑现。

#### ② 光学板块获利了结（~30%）

| 背景 | 说明 |

→ **没有三家同时爆雷**，是 **positioning / profit-taking**。

#### ③ 6/15 预测兑现：「不追、防高开回落」（~20%）

6/15 文档 §5.1 基准情景（~55%）已写：

> 典型走法：高开 → 获利盘 → 收盘 ±2%～±4%，AAOI 波动 > COHR > LITE

#### ④ 宏观次要因素（~15%）

- **Fed 6/16–17 议息**：高估值成长在会议前常减仓  
- **SpaceX 首日暴涨后分化**：风险偏好不扩散至全线

## 11. 联动标的：SMCI 为何也跌很多？

> SMCI 与 LITE/COHR/AAOI **同属 AI 基建高 Beta 组**，但 **下跌主因不同**——不宜混为一谈。

### 11.1 SMCI 跌幅（Yahoo，6/16 盘中 ~$29.69）

| 窗口 | 跌幅 | 说明 |
| --- | ---: | --- |
| **6/10 单日** | **-28.0%** | $40.64 → $29.27，**主跌日** |
| 6/9 → 6/16 累计 | **-26.9%** | 近一周 |
| 52W 高 $62.36 → 现价 | **-52.4%** | 远超大盘 |

6/16 相对 6/12（$30.46）仅 **~-2.5%**——SMCI 的重伤发生在 **6/10**，不是 6/16 光学回调那天。

### 11.2 主因：$70 亿融资 / 稀释恐慌（公司特有）

6/9–6/11 SMCI 宣布并通过定价：

| 项目 | 内容 |
| --- | --- |
| 融资规模 | 最高 **$70 亿**（$50 亿承销 + $20 亿 ATM + 可选超额） |
| 用途 | 采购零部件，履行 **$390 亿** AI 服务器订单（20+ 客户） |
| 市场反应 | 稀释估算 **26%–35%** 股本 → 6/10 暴跌 |
| 配套利空 | 库存 **$111 亿**、FCF **-$67 亿**、EY 辞任审计、DOJ 调查等治理阴影 |

**逻辑链：**

```text
$390B 订单（利好）→ 需 $70B 股权融资（稀释）→ 市场先罚 dilution，后等 execution
```

这与光模块 **「和谈卖事实 + 获利了结」** 完全不同——SMCI 是 **「good news is bad news」** 经典案例。

### 11.3 与光模块回调的关系

**弱联动：** 同属「AI 基建 trade」，资金有时会 **一并减仓高 Beta**；但 SMCI 跌幅 **90% 来自 6/10 融资事件**，不是 6/16 光学带跌。
