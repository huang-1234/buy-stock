# SMCI：2026 年 6 月估值合理性（偏高 / 偏低 / 合理）

AI 产业链 **L2**（AI 服务器/机柜）。数据截至 **2026 年 6 月中旬**（~$31），供研究参考，**非投资建议**。

> 贝叶斯 intrinsic：[ana.md](./ana.md) · Q4 情景分析：[q4-scenario-aug2026.md](./q4-scenario-aug2026.md)

---

## 1. 结论（TL;DR）

| 维度 | 判断 |
| --- | --- |
| **绝对估值（盈利质量）** | **仍偏贵** |
| **名义股价位置** | **从极端高估大幅回落**（52W 高 $62 → ~$31，**-50%**） |
| **intrinsic vs 市场（收入）** | ~35% vs ~38%–45%，gap **−5pp** |
| **盈利 / 现金流** | GM **~10%**、OCF **负**、稀释 **~25%–30%** → **质量严重滞后** |
| **估值状态** | **高估但可交易 / 高风险** |
| **仓位** | 观察 / 卫星 **≤5%** |

**一句话：** ~$31 不是 fair value，是 **高风险折价** — 赌 Q4 兑现 margin/OCF，不是买「便宜 AI 股」。

## 3. 为何「名义价低、质量估值仍偏贵」

### 3.1 收入维度：gap −5pp，略偏贵

市场仍按 FY26 **~$40B** 收入定价，intrinsic **~35%** 低于隐含 **~38%–45%**，但 **置信度低**（Q3 site delay、execution 历史）。

### 3.2 盈利维度：明显偏贵

| 视角 | 判断 |
| --- | --- |
| P/S ~0.74× | 像便宜 |
| GM 10% → 8% 指引 | 赚「流水」不赚「利润」 |
| OCF 大幅为负 | 价值靠融资维持 |
| 稀释 ~30% | 每股 earnings power 被稀释 |

ana.md 定性：**「故事领先、质量严重滞后」**。

### 3.3 股价位置 vs 合理价

| 对比 | 关系 |
| --- | --- |
| vs 52W 高 $62 | **名义偏低** |
| vs intrinsic 收入故事 | **仍略偏贵** |
| vs 分析师 PT ~$39–44 | 有 upside，beat 历史差 |
| vs DELL/HPE 等同业 | AI 溢价 **在收缩** |

## 4. 为何没跟着 NVDA / GOOGL 涨

| 在涨的 | SMCI 没涨的原因 |
| --- | --- |
| NVDA — GPU 毛利 ~75% | SMCI — 组装 GM **~10%** |
| GOOGL — Cloud margin 32.9% | SMCI — 只有 revenue，无 margin 故事 |
| MU — memory 周期 | SMCI — **6/10 $70 亿增发** 稀释优先于 $390B backlog |
| | Q3 **revenue miss 18%**，执行信用受损 |

6 月资金从 **「AI 收入叙事」** 切到 **「AI 利润/ROI 叙事」** — SMCI 落在被惩罚一侧。

## 8. 大盘回落敏感性（6/16 前后）

> 背景：6/15 risk-on 反弹后，SMCI 从 ~$29 弹至 ~$31。大盘回落日 **高 Beta 资产往往跌得更猛**。参见 [光模块 6/15 判断](../../optional-fiber-communications/jun15-2026-us-rally.md) §5（基准情景：涨后整理 / 小跌 ~55%）。

### 8.1 结论

| 问题 | 判断 |
| --- | --- |
| 大盘回落会不会跌？ | **会**，概率更高 |
| 相对大盘 | 约 **1.5×–3×** SPX 跌幅 |
| vs MSFT | **跌得更猛**（高 Beta + 趋势 Broken） |

### 8.2 为何放大下跌

- **高 Beta、高波动**（6/10 单日 **-28%** 含增发；纯大盘日亦常放大）  
- GF-DMA 趋势 **Broken**，无机构护城河  
- 6/15 反弹为 **情绪修复**，非 Q4 新 catalyst  
- 距 8/4 财报远，**获利盘先走「脏 L2」**  

### 8.3 情景估算

| 大盘（SPX） | SMCI 典型反应 |
| --- | --- |
| **-0.5% ~ -1%** | **-1.5% ~ -3%** |
| **-1% ~ -2%** | **-3% ~ -6%** |
| **-2%+ risk-off** | **-5% ~ -10%**，考验 **$29** |

**参照：** 6/10 大盘 SPX **-1.6%** 时 SMCI **-28%** 含 **$70 亿增发** 个股因素；纯大盘回落日通常为 Beta 放大，非极端单日。

### 8.4 回落类型

| 类型 | 触发 | SMCI 表现 |
| --- | --- | --- |
| **A 温和获利了结** | SPX -0.5% ~ -1.5% | -2% ~ -4%，中等阴线 |
| **B risk-off** | 地缘 / 油价 / 通胀 | -5% ~ -8% |
| **C AI capex 恐慌** | Hyperscale 削减指引 | L2 需求叙事受损，**连带重挫** |

### 8.5 支撑

| 支撑 | 跌破后 |
| --- | --- |
| **$29–$30**（6/10 低） | $27 → 52W 低 **$19.48** 区间 |

跌破 **$29** → 增发后结构再受考验，**不宜盲目抄底**。

### 8.6 独立不跌概率

**~15%–20%**，需突发 backlog/订单新闻（8/4 前少见）。

### 8.7 回落日操作

| SPX | 建议 |
| --- | --- |
| -0.5% ~ -1.5% | 卫星仓可持有，**不加仓** |
| -2%+ | 盯 **$29**；破则减至 lottery 或观望 |
| 类型 C | 等待 8/4 Q4 再评估 |

### 8.8 vs MSFT（同日对照）

| | MSFT | SMCI |
| --- | --- | --- |
| 相对 SPX | 0.9×–1.2× | 1.5×–3× |
| 抗跌性 | 较强 | 弱 |
