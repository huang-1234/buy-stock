# MSFT：2026 年 6 月估值合理性（偏高 / 偏低 / 合理）

AI 产业链 **L3/L5**（Azure/Copilot/企业 AI）。数据截至 **2026 年 6 月中旬**（~$400），供研究参考，**非投资建议**。

> 贝叶斯 intrinsic：[ana.md](./ana.md) · GF-DMA / 反弹论证：[gf-dma-jun2026.md](./gf-dma-jun2026.md)

---

## 1. 结论（TL;DR）

| 维度 | 判断 |
| --- | --- |
| **绝对估值** | **合理，略偏贵** |
| **intrinsic vs 市场** | ~22% vs ~22%–26%，gap **−2pp** |
| **相对 Azure +40% 基本面** | **情绪略偏低**（R_50 < 0.5） |
| **L3 内对比** | 次于 GOOGL；优于 SMCI |
| **仓位** | 中仓 **15%–20%** |

**一句话：** 基本面配得上大部分溢价，但 ~$190B capex 压制 re-rating；6 月回调后 **不是 deep value**，而是 **quality 龙头的情绪 undershoot**。

---

## 3. 为何「合理略偏高」而非「便宜」

1. **gap 仅 −2pp** — intrinsic 与 market 对齐，无显著安全边际  
2. **Forward P/E ~25–26×** — 大盘 quality 成长正常区间，非历史低位  
3. **CapEx 叙事未缓和** — Q3 beat 后仍因 $190B 指引持续阴跌  
4. **L3 内 GOOGL 更优** — 同 capex 量级，GOOGL 有 Cloud margin **32.9%** 背书，MSFT 无单独 Cloud 利润披露  

---

## 4. 为何又说「相对基本面略偏低」（情绪）

| 信号 | 状态 |
| --- | --- |
| R_50 = G_DMA50 / G_f | **< 0.5** |
| 6/1 → 6/11 | $460 → $386（**-13%**） |
| Azure 增速 | **+40%** 未恶化 |
| 解读 | 股价跌得 **比基本面「该跌的」更多** |

**含义：** 支持向 **50DMA（~$412）** 的 mean reversion（+5% ~ +9%），但 **不等于绝对估值偏低**。

## 5. 与 AI 链 peers 对比（6 月）

| 标的 | 绝对估值 | 6 月股价 | 市场在定价 |
| --- | --- | --- | --- |
| GOOGL | 合理 | 抗跌（~-1.5%） | Cloud +63% + margin 可见 |
| NVDA | 合理偏高 | YTD 强 | GPU 定价权 |
| **MSFT** | **合理略偏高** | **回调 -13%** | 收入强，FCF 故事弱 |

**MSFT 没涨主因：** Q3 beat 后 market 惩罚 **CapEx $190B**（超 consensus ~22%）；6/11 Oracle cloud miss 连带 hyperscaler；同等 capex 下 **GOOGL 被奖励、MSFT 被惩罚**（ROI 证据差异）。
